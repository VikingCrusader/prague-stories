Ark Pixel Font (方舟像素字体) — Required for Chinese pixel-style text
=====================================================================

Download steps:
1. Go to: https://github.com/TakWolf/ark-pixel-font/releases/latest
2. Download the file named:
     ark-pixel-font-12px-proportional-zh_cn-woff2-*.zip
3. Extract the zip — you will find a .woff2 file inside.
4. Rename it to:  ark-pixel-12px.woff2
5. Place it in this folder:  client/public/fonts/

The app will automatically load it for the ZH (Chinese) language mode.
ZCOOL QingKe HuangYou (Google Fonts) is used as fallback until the file is placed here.
